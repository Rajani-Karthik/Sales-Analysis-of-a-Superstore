🛍️ Sales Analysis of Superstore (Tableau Dashboard)
📌 Overview

This project presents an interactive Tableau dashboard analyzing sales data from a retail superstore. It provides insights into sales performance, profitability, customer segments, and regional trends to support data-driven decision-making.

🎯 Objectives
Analyze overall sales and profit performance
Identify top-performing products and categories
Understand regional sales distribution
Evaluate customer segments and buying behavior
Detect trends and patterns over time
🛠️ Tools & Technologies
Tableau Desktop
Data Visualization
Business Intelligence
